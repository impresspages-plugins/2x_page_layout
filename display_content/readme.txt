ImpressPages CMS plugin


INSTALL

1. Upload folder "display_content" to "ip_plugins" directory of your website.
2. Login to administration area
3. Go to "Developer -> Modules" and press "install".
4. Visit your website. You should see a new tab under Display content called Page layouts, go here and add a new record, choose from available layouts and selects pages its assigned to.
5. If you would like to exclude some zones from the page list you can do this with Standard / Configuration of Page layouts and add a list of zones key names on seperate lines (remember key names are lowercase)
